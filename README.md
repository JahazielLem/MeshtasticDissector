# MeshtasticDissector V0.0.1
Wireshark dissector for LoRa Meshtastic

Work in progress:
- Messages Text fields
- Postion fields
- Add encryption key handler

# Current fields supported
- **Core**:
  - [x] Bitfield
  - [x] Portnum
- **Text Message**:
  - [x] Length
  - [x] Emoji
  - [x] Text
- **Position**:
  - [x]  Latitude
  - [x]  Longitude
  - [x]  Timestamp


Stable dissector with:
- Basic packet field
- Key preferences list
